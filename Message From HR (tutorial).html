Welcome to Gone Phishing

You've been hired by a large tech company to help with communication. While you work you may run into some technical diffuculties and issues. This game is to help you learn about the proper way to handle those situations.

Files like this one are html files and will be used for expositions and explanations.

The rest of the files are link files, they are your actions double click on whatever option you want and it will take you to a new part of the game.

Please enjoy and we hope you learn something.
